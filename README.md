## Instalacja

1.  **Klonuj repo:**

    ```bash
    git clone <URL>
    cd photoalbum-backend
    ```

2.  **Stworz venv:**
   
    ```bash
    python -m venv .venv
    ```

3.  **Instaluj wymagania:**
    
    ```bash
    pip install -r requirements.txt
    ```

4.  **Odpal:**

    ```bash
    python manage.py runserver
    ```
